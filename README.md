### ShopCart Ecommerce Backend

## Project Setup

1. Clone the project

```
git clone https://github.com/iitiansandy/ShopCart-Ecommerce.git
```
2. Move into the directory of the cloned project

```
cd ECom-Backend
```

3. Install node_modules

```
npm i OR npm install
```

4. Turn up the server

```
npm start
```
